# SunoPlay 🎵

SunoPlay es una aplicación web premium diseñada para explorar y escuchar las mejores creaciones musicales de **Suno AI**.

## Características
- **Streaming Directo**: Se conecta directamente a la CDN de Suno para reproducir audio de alta calidad.
- **Diseño Premium**: Interfaz moderna con efectos de glassmorphism, modo oscuro y animaciones fluidas.
- **Explorador de Tendencias**: Lista seleccionada de canciones que son tendencia en la plataforma.
- **Reproductor Completo**: Control de reproducción, barra de progreso, volumen y visualización de carátulas.
- **Búsqueda Inteligente**: Filtra canciones por título, artista o estilo musical.

## Cómo Ejecutar
1. Simplemente abre el archivo `index.html` en cualquier navegador moderno.
2. Explora las canciones en la cuadrícula o usa el botón "Reproducir Ahora" en el banner principal.
3. ¡Disfruta de la mejor música generada por IA!

## Detalles Técnicos
- **Frontend**: Vanilla HTML5, CSS3 y JavaScript.
- **Iconos**: Lucide Icons.
- **Tipografía**: Outfit (vía Google Fonts).
- **Audio**: Streams directos desde `cdn1.suno.ai`.

*Nota: Esta aplicación es un reproductor independiente que utiliza contenido público de Suno.com.*
